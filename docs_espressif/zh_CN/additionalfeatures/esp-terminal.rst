ESP-IDF 终端
================

:link_to_translation:`en:[English]`

1.  前往菜单栏 ``查看`` > ``命令面板``。
2.  输入 ``ESP-IDF：打开 ESP-IDF 终端`` 并选中该命令。这将启动系统终端，并将 ESP-IDF、ESP-IDF 工具以及 ESP-IDF Python 虚拟环境作为环境变量加载。
3.  输入 ``idf.py`` 或 ``python -m esptool`` 即可运行 ESP-IDF 及其他附加框架的脚本。
